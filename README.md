## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/MD-KAIF16/KAIF_PORTFOLIO.git
```

Move into the project directory:

```bash
cd KAIF_PORTFOLIO
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
